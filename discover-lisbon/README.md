## 🚀 Tech Stack

| Tecnologia | Porquê |
|---|---|
| [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) | Build rápido e moderno |
| TypeScript | Segurança de tipos |
| Vanilla CSS | Controlo total, glassmorphism |
| [Framer Motion](https://www.framer.com/motion/) | Animações suaves |
| [Lucide React](https://lucide.dev/) | Ícones modernos |

---

## 📦 Como correr localmente

```bash
cd discover-lisbon
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173)

## 🔨 Build de produção

```bash
npm run build
```

## 📁 Estrutura

```
src/
  components/
    Hero.tsx       — Secção principal
    About.tsx      — História e valores
    Schedule.tsx   — Horários de treino
    Club.tsx       — Estrutura do clube
    Community.tsx  — Vida social
    Gallery.tsx    — Galeria & Instagram
    Shop.tsx       — Loja de merch
  App.tsx          — Navegação + Contacto + Footer
  index.css        — Design system global
index.html         — SEO meta tags
```

---

## 🎨 Design

- Tema escuro com **glassmorphism**
- Paleta: `#0f172a` (fundo) · `#f6ad55` (accent laranja) · `#f8fafc` (texto)
- Tipografia: **Outfit** (títulos) + **Inter** (corpo)
- Totalmente responsivo (mobile-first)
